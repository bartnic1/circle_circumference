### Circle Math

## Quickstart
```
npm i circle_math

const circles = require('circle_math');

console.log(circles.circle_area(radius))
```
::or::

```
console.log(circles.circle_circumference(radius))

returns the circumference of a circle with a given radius or the area!!!
```
